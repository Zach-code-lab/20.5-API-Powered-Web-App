# Pokémon Viewer

## API Used
PokéAPI (https://pokeapi.co)

## What My App Does
You type in a Pokémon name and the app fetches real-time data from the PokéAPI. It displays the Pokémon’s name, height, weight, base XP, and an image.

## What Challenged Me Most
The hardest part was organizing the code using the Data → Logic → UI pattern. Once I separated the fetch call, the data processing, and the DOM updates, everything made more sense.
